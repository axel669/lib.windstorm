Hexagon Spiner

The hexagon spinner is an element that shows an animated hexagonal load spinner.

Wind Functions

  • $color
    Sets the colors for each of the hexagons.

Variables

  • size
    The width and height of the spinner. The animated pieces fit into the size given.
<ws-hexagon-spinner></ws-hexagon-spinner>
<ws-hexagon-spinner ws-x="&size[50px] $color[warning]"></ws-hexagon-spinner>
<ws-hexagon-spinner ws-x="&size[200px] $color[accent]"></ws-hexagon-spinner>
<ws-hexagon-spinner ws-x="&size[300px] $color[danger]"></ws-hexagon-spinner>
