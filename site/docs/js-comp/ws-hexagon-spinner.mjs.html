Hexagon Spiner

The hexagon spinner is an element that shows an animated hexagonal load spinner.

Wind Functions

  • $color
    Sets the colors for each of the hexagons.

Variables

  • size
    The width and height of the spinner. The animated pieces fit into the size given.
