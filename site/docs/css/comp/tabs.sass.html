Tabs

Displays a list of tabs with some default styling. Tabs do not respond to inputs without JavaScript.

Use the tab-selected attribute to show which tab is active.

Wind Functions

  • @solid
  • $color

    </ws-titlebar>
    <div ws-x="$content">
        <ws-tabs ws-x="$color[secondary] w[50%]">
            <ws-tab>Light</ws-tab>
            <ws-tab>Dark</ws-tab>
            <ws-tab tab-selected>Tron</ws-tab>

</ws-paper>
