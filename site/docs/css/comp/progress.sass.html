Progress Bar

The progress bar is a modified label element that changes how the label and progress html element look.

To mark a label as a Progress Bar, add the @progress wind function.

Wind Functions

  • @outline
  • $color
  • $row

Areas

  • $text

Preview

Code Example

<label ws-x="@progress @outline">
    <span ws-x="$text">Some progress</span>
    <progress value="0.3"></progress>
</label>
<label ws-x="@progress">
    <span ws-x="$text">Some progress (flat)</span>
    <progress value="0.2"></progress>
</label>
<label ws-x="@progress @outline $color[secondary]">
    <span ws-x="$text">Some color progress</span>
    <progress value="0.6"></progress>
</label>
<label ws-x="@progress @outline $row $color[accent]">
    <span ws-x="$text t-ws[nowrap]">Row Layout</span>
    <progress value="0.8"></progress>
</label>

No Text Label
<label ws-x="@progress @outline $color[primary]">
    <progress value="0.9"></progress>
</label>
