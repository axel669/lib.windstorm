Notification

Displays some text in an area intended to draw attention towards it either inline or in conjuction with the toaster component.

Wind Functions

  • $color
<ws-notification>
    <ws-icon class="ti-info-hexagon">Pretend has useful info</ws-icon>
</ws-notification>
<ws-notification ws-x="$color[primary]">
    <ws-icon class="ti-info-hexagon">This info has color</ws-icon>
</ws-notification>
<ws-notification ws-x="$color[warning]">
    <ws-icon class="ti-help-hexagon">You can add controls</ws-icon>
    <button ws-x="@fill $color[warning]">Click!</button>
</ws-notification>
<ws-notification ws-x="$color[danger]">
    <button ws-x="@fill $color[danger]">Click!</button>
    <div>
        And reverse the order
        <ws-icon class="ti-alert-hexagon"></ws-icon>
    </div>
</ws-notification>
