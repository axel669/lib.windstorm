Titlebar

Displays a titlebar inside a screen or card (or anywhere really, but it looks pretty cool with those). Allows customization of the text and side controls.

Wind Functions

  • @fill
  • $color
<ws-titlebar>
    <span ws-x="slot[title]">
        <div ws-x="$title">Basic Titlebar</div>
        <div ws-x="$subtitle">Subtitle</div>
    </span>
</ws-titlebar>

<ws-titlebar ws-x="$color[primary]">
    <span ws-x="slot[title] $title">Color Titlebar</span>
</ws-titlebar>

<ws-titlebar ws-x="$color[danger]">
    <span ws-x="slot[title] $title">Titlebar + controls</span>

    <button ws-x="@flat slot[menu]">
        <ws-icon class="ti-menu-2"></ws-icon>
    </button>
</ws-titlebar>

<ws-titlebar ws-x="@fill $color[primary]">
    <span ws-x="slot[title] $title">Filled Color Titlebar</span>
</ws-titlebar>

<ws-titlebar ws-x="@fill $color[secondary]">
    <span ws-x="slot[title] $title">Filled Titlebar + controls</span>
    <button ws-x="@flat slot[action]">
        <ws-icon class="ti-logout"></ws-icon>
    </button>
</ws-titlebar>
