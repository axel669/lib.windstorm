Badge

Adds a small badge in the upper-right corner of some content.
Reads the text from the ws-text attribute.

Wind Functions

  • $color

Preview

Code Example

<div ws-x="h[24px]"></div>
<ws-badge ws-text="0">
    <div ws-x="p[8px] bd[1px_solid_var(--primary)]">
        <ws-icon class="fa-message">
            Messages
        </ws-icon>
    </div>
</ws-badge>
<ws-badge ws-text="text">
    <div ws-x="p[8px] bd[1px_solid_var(--primary)]">
        <ws-icon class="fa-message">
            I Dunno
        </ws-icon>
    </div>
</ws-badge>
