Labled Control

The labeled control is a label that contains a text-style input (input that isn't checkbox or radio, textarea) or a select element and the text that make up the label.

To mark a label as a labeled control, add the @control wind function.

Wind Functions

  • @flat
  • $color

Children Wind Functions

  • $text
    sets the element with the text for the label
  • slot[start / end]
    adds an adornment to the input area, probably only looks good with input elements that are typed text.
<ws-grid ws-x="gr-col[1fr_1fr]">
    <label ws-x="@control $color[primary]">
        <span ws-x="$text">OOP Language</span>
        <input />
    </label>
    <label ws-x="@control">
        <span ws-x="$text">Cheese Picture</span>
        <input type="file" />
    </label>
    <label ws-x="@control $color[secondary]">
        <span ws-x="$text">Coolness Factor</span>
        <input />
        <div ws-x="$adorn[start]">
            $
        </div>
    </label>
    <label ws-x="@control @flat $color[warning]">
        <span ws-x="$text">Mistakes in CS</span>
        <select>
            <option>TypeScript</option>

        </select>
    </label>

    <label ws-x="@control col[span_2] $color[danger]">
        <div ws-x="$text">TextArea</div>
        <textarea ws-x="h[100px]"></textarea>
    </label>
</ws-grid>
