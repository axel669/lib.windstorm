Flex

Flex container with a few nice presets to space the content and allow scrolling to be added easily.

Preview

Code Example

Default Direction (column)
<ws-flex>
    <ws-chip ws-x="@outline $color[primary]">
        Item 1
    </ws-chip>
    <ws-chip ws-x="@outline $color[primary]">
        Item 2
    </ws-chip>
    <ws-chip ws-x="@outline $color[primary]">
        Item 3
    </ws-chip>
    <ws-chip ws-x="@outline $color[primary]">
        Item 4
    </ws-chip>
</ws-flex>
