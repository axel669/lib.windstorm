Labled Control

The labeled toggle is a label that contains a toggle-style input (type "checkbox"/"radio"). A checkbox input can also have the $switch wind function added to visual change it into a switch toggle.

To mark a label as a labeled toggle, add the @toggle wind function.

Wind Functions

  • @outline
  • $color

Children Wind Functions

  • $switch
    Turns a checkbox into a different visual style of input
<ws-paper ws-x="@outline">
    <ws-titlebar ws-x="$header">
        <span ws-x="$title">Checkbox</span>
    </ws-titlebar>
    <ws-flex ws-x="$content">
        <label ws-x="@toggle">
            <span ws-x="$text">I'm running out of label ideas</span>
            <input type="checkbox" />
        </label>
        <label ws-x="@toggle $color[danger]">
            <span ws-x="$text">What do I even put here</span>
            <input type="checkbox" />
        </label>
    </ws-flex>
</ws-paper>

<ws-paper ws-x="@outline">
    <ws-titlebar ws-x="$header">
        <span ws-x="$title">Switch</span>
    </ws-titlebar>
    <ws-flex ws-x="$content">
        <label ws-x="@toggle $color[accent]">
            <span ws-x="$text">Why is documentation so tedious</span>
            <input type="checkbox" ws-x="$switch" />
        </label>
        <label ws-x="@toggle @outline $color[primary]">
            <input type="checkbox" ws-x="$switch" />
            <span ws-x="$text">Reverse reverse</span>
        </label>
    </ws-flex>
</ws-paper>
