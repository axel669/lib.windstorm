Link

Styling for the a tag. Tags without the ws-x attribute won't be styled, while the attribute with no functions will still style it. Links can be presented as link buttons by adding the @button wind function.

Wind Functions

  • @button
    • @outline or @fill (with @button only)
  • $color
<ws-flex ws-x="fl-cr-a[start]">
    <a href="https://github.com" ws-x>GitHub</a>
    <a href="https://github.com" ws-x="$color[primary]" target="_blank">GitHub</a>
    <a href="https://github.com" ws-x="@button" target="_blank">GitHub</a>
    <a href="https://github.com" ws-x="@button $color[warning]">GitHub</a>
    <a href="https://github.com" ws-x="@button $color[primary] @outline" target="_blank">GitHub</a>
    <a href="https://github.com" ws-x="@button $color[secondary] @outline">GitHub</a>
    <a href="https://github.com" ws-x="@button $color[danger] @fill w[120px]">GitHub</a>
    <a href="https://github.com" ws-x="@button $color[accent] @fill h[75px]" target="_blank">GitHub</a>
</ws-flex>
